# 🤖 Bot WhatsApp Rifa com Baileys

Bot de Rifa WhatsApp feito com Node.js usando a biblioteca Baileys.

---

## 🚀 Funcionalidades

* Conexão com WhatsApp via QR Code
* Resposta automática de mensagens
* Integração com MongoDB

---

## 📦 Tecnologias

* Node.js
* Baileys
* MongoDB
* Mongoose

---

## ⚙️ Instalação

```bash
git clone https://github.com/seu-usuario/seu-repo.git
cd seu-repo
npm install
```

---

## 🔑 Configuração

Crie um arquivo `.env`:

```
PORT=3000
MONGO_URI=sua_string_do_mongodb
```

---

## ▶️ Rodar o projeto

```bash
node server.js
```

---

## 📱 Conexão

* Escaneie o QR Code no terminal
* Aguarde a conexão

---

## ⚠️ Observações

* Se der erro, apague a pasta:

```bash
rm -rf bot/auth_info
```
